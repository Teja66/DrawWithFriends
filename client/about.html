About

Draw with friends is a collaborative drawing tool. Using this friends can draw together real time. This project demonstrates how to build a real-time communication channel between one server and multiple clients by using Node.js, Socket IO, Express, JavaScript, HTML5, CSS3 and JQuery. The express library is used to power the web server and send the files in the client folder to the clients. Each user can type in a room name and connect. User can see or modify the contents of the canvas. User can choose colors, brush size and can also save the drawings. Try it..and have fun!!
